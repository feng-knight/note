centos下安装MongoDB

1.下载安装 MongoDB

2. 创建文件夹和解压文件

  • cd 到 usr 目录

  • 新建mongodb目录

  • 解压到usr/mongodb 目录下

  • 重命名文件夹

3.配置环境变量和初始化操作

  • 配置环境变量

    在 export PATH USER LOGNAME MAIL HOSTNAME HISTSIZE HISTCONTROL上面一行添加export PATH=/usr/mongodb/mongodb-4.0.10/bin:$PATH

  • 使环境变量生效

  • 创建mongodb.conf配置文件

  • 在该目录下创建数据库目录，日志目录

  • 设置文件夹权限，方便操作

  • 创建日志 文件

4.修改配置文件内容

  • 添加mongodb.conf配置文件

5.启动mongodb

可能出现的问题:

原因是可能27017端口被占用！！

解决方案：先使用netstat -ntlp 查看mongo.conf配置文件里对应的 port=27017， 然后使用kill -9 14540 【PID进程号】 杀死此进程 最后在到bin目录下再次执行

出现上述信息则代表配置文件成功配置，数据库成功打开!

6.关闭防火墙

  • 关闭防火墙

7. 开心的连接数据库啦！

在浏览器中使用 公网ip/域名：27017如果看到如下的信息则代表成功开启mongodb数据库！
