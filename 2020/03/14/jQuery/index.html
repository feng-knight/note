jQuery

一、jQuery：（一个快速、简洁的JavaScript库），J:JavaScript，Query：查询js

  1. jQuery的入口函数：

    $( function( ) { } ); 或 $(document).ready( function( ) { } ); //此处是页面DOM加载完成的入口

  2. jQuery的顶级对象$：（jQuery = $）

  3. jQuery对象：

    • 获取和使用对象：

      $(‘div’)；和 $(‘div’).方法；

    • DOM对象转换为jQuery对象：

      $(DOM对象)；

    • jQuery对象转换为DOM对象：

      $(‘div’)[index]; 或 $(‘div’).get(index); //index是索引号

  1. jQuery选择器：

    • jQuery基础选择器：$(“选择器”)；

      名称 用法 描述
      ID选择器 $(“#id”) 获取指定元素Id
      全选选择器 $(“*”) 匹配所有元素
      类选择器 $(“.class”) 获取同一类class的元素
      标签选择器 $(“div”) 获取同一类标签的所有元素
      并集选择器 $(“div,p,li”) 选取多个元素
      交集选择器 $(“li.current”) 交集元素

    • jQuery层级选择器：

      名称 用法 描述
      子代选择器 $(“ul>li”); 使用>号，获取亲儿子层级的元素;注：并不会获取孙子层级的元素
      后代选择器 $(“ul li”); 使用空格，代表后代选择器，获取ul下的所有li元素，包括孙子等

    • 隐式迭代：（遍历循环内部DOM元素(伪数组形式存储)的过程)）

    • jQuery筛选选择器：

      语法 用法 描述
      :first $(‘li:first’) 获取第一个li元素
      :last $(‘li:last’) 获取最后一个li元素
      :eq(index) $(‘li’:eq(2)) 获取到的li元素中，选择索引号为2的元素（索引号index从0开始）
      :odd $(‘li:odd’) 获取到的li元素中，选择索引号为偶数的元素
      :even $(‘li:even’) 获取到的li元素中，选择索引号为奇数的元素

    • jQuery筛选方法：（重点）

      语法 用法 说明
      parent()和parents(“.类名”) $(“li”).parent();和$(“li”).parent(“.current”); 查找父级和返回指定的祖先级元素
      children(selector) $(“ul”).children(“li”); 相当于$(“ul>li”)，最近一级（亲儿子）
      find(selector) $(“ul”).find(“li”); 相当于$(“ul li”)，后代选择器
      siblings(selector) $(“.first”).siblings(“li”); 查找兄弟结点，不包括自己本身
      nextAll( [expr] ) $(“.first”).nextAll(); 查找当前元素之后所有的同辈元素
      prevtAll( [expr] ) $(“.last”).prevtAll(); 查找当前元素之前所有的同辈元素
      hasClass(class) $(“div”).hasClass(“protected”); 查找当前的元素是否含有某个特定的类，如果有，返回true
      eq(index) $(“li”).eq(2); 相当于$(“li:eq(2)”)

    • jQuery得到当前元素索引号：$(this).index();

  2. jQuery样式操作：

    • 操作css方法：

      1) $(this).css(‘color’); // 返回属性值

      2) $(this).css(‘color’，’red’); // 修改样式属性值内容

      3) $(this).css(‘color’：’blue’，’font-size’：’20px’); // 修改多组样式属性值内容

    • 操作类样式方法：

      • 添加类：$(‘div’).addClass(‘current’); // 注：里面的类千万不要加”.”
      • 移除类：$(‘div’).removeClass(‘current’);
      • 切换类：$(‘div’).toggleClass(‘current’); // 若有此类则移除此类；反之添加此类

  3. jQuery动画效果：

    • 显示隐藏：show()、hide()、toggle()—>切换

      1) show( [speed]，[easing]，[fn] ] )

      2) hide( [speed]，[easing]，[fn] ] )

      3) toggle( [speed]，[easing]，[fn] ] )

      • 参数可以省略，无动画直接显示

      • speed：三种预定速度之一的字符串（“slow”，“normal”，“fast”）或 表示动画时长的毫秒数

      • easing：用来指定切换效果，默认是“swing”，可用参数“linear”

      • fn：回调函数，在动画完成时执行的函数，每个元素执行一次

    • 滑动：slideDown()、slideUp()、slideToggle()【所有参数同上】

    • 淡入淡出：fadeIn()、fadeOut()、fadeToggle()、fadeTo()【所有参数同上】

      fadeTo( [speed]，opacity，[easing]，[fn] ) //调整透明度

      • opacity 透明度必须写，取值在0~1之间
      • speed 也必须写

    • 自定义动画：animate()

      animate( params，[speed]，[easing]，[fn] )

      params：想要更改样式的属性，以对象形式传递(对象里不带单位！)，必须写。属性名不可以带引 号，如果是复合属性则需要采取驼峰命名法borderLeft。其余参数都可以省略

    • 停止动画：stop() //写到动画效果的前面，相当于停止上一次动画—>.stop() .slideToggle();☆☆☆☆☆

    • 事件切换（鼠标经过和离开）：hover()

      hover( [over]，[out] )

      • over：鼠标移到元素上要触发的函数（相当于mouseenter）
      • out：鼠标移除元素要触发的函数（相当于mouseleave）
      • 若只写一个函数，那么鼠标经过和鼠标离开都会触发这个函数

  4. jQuery属性操作：

    • 获取和设置元素固有属性值：prop(“属性”); 和 prop(“属性”,”属性值”);

    • 获取和设置自定义属性：attr(“属性”); 和 attr(“属性”,”属性值”); //类似原生get/setAttribute()

    • 数据缓存：data(“name”); 和 data(“name”,”value”); //数据存放在元素的内存中，DOM中看不见此数据

      注：这个方法获取data-index h5自定义属性 第一个 不用写data- 而且返回的是数字型

  5. jQuery内容文本值：

    • 普通元素内容：html() (相当于原生innerHTML)

      1) html() //获取元素的内容

      2) html(“内容”) //设置元素的内容

    • 普通元素文本内容：text() (相当于原生innerText)【方法同上】

    • 表单的值：val() (相当于原生value)【方法同上】

  6. jQuery元素操作：

    • 遍历元素：each()

      1) $(“div”).each( function ( index，domEle ) { } );

      • index：每个元素索引号
      • domEle：每个DOM元素对象，不是jQuery对象

      2) $.each( object，function( index，element ) { } );

      • $.each()方法可用于遍历任何对象。主要用于数据处理，比如数组、对象
      • element：遍历内容

    • 创建元素：$(“

    • 内容
      • “)

    • 添加元素：

      1) 内部添加：

      • element.append(“内容”) //放在后面

      • element.prepend(“内容”) //放在前面

      2) 外部添加：

      • element.after(“内容”) //放在后面

      • element.before(“内容”) //放在前面

    • 删除元素：

      1) element.remove() //删除匹配的元素（本身）

      2) element.empty() //删除匹配的元素集合中所有的子节点

      3) element.html(“”) //删除匹配的元素集合中所有的子节点

  1. jQuery事件处理：

    • on() 绑定事件：（该方法在匹配元素上绑定一个或多个事件的事件处理函数）

      element.on( events，[selector]，fn )

      • events：一个或多个用空格分隔的事件类型，如“click”或“keydown”
      • selector：元素的子元素选择器

      eg：

      或

    • one() 绑定事件：(如果有的事件只想触发一次可以用此方法)，使用方法与on()一致

    • off() 解绑事件：

      1) $(“div”).off(); // 这个是解除了div身上的所有事件

      2) $(“div”).off(“click”); // 这个是解除了div身上的点击事件

      3) $(“ul”).off(“click”, “li”); //解绑事件委托

    • 自动触发事件：

      1) element.click()

      2) element.trigger(“事件”)

      3) element.triggerHandler(“事件”) //此方法不会触发元素的默认行为

  2. jQuery事件对象：

    • 阻止默认行为：event.preventDefault() 或 return false
    • 阻止冒泡：event.stopPropagation()

  3. jQuery对象拷贝：

    • $.extend( [deep]，target，object1，[objectN] )
      • deep：true为深拷贝，默认是false
      • target：要拷贝的目标对象
      • object1：待拷贝到第一个对象的对象

  4. jQuery多库共存：

    • $变为jQuery
    • $.noConflict()：释放控制权给用户自定义（var suibian=$.noConflict()）

  1. jQuery尺寸：

    语法 用法
    width() / height() 取得匹配元素宽度和高度值(只算width / height)
    innerWidth() / innerHeight() 取得匹配元素宽度和高度值(包含padding)
    outerWidth() / outerHeight() 取得匹配元素宽度和高度值(包含padding、border)
    outerWidth(true) / outerHeight(true) 取得匹配元素宽度和高度值(包含padding、border、margin)

  2. jQuery位置：

    • offset()：设置或获取元素偏移

      • 设置或返回相对于文档的偏移坐标，跟父级没有关系
      • 两个属性：left、top【offset().top】
      • 设置元素的偏移：offset(｛top：10，left：30｝);

    • position()：获取元素偏移（返回被选元素相对于带有定位的父级偏移坐标，若父级都没有定位则以文档为准）

    • scrollTop() / scrollLeft()：设置或获取元素被卷去的头部和左侧
