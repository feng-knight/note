一、JS基础：

  1. JavaScript:

    • ECMAScript(JavaScript语法)
    • DOM（文档对象模型）
  • BOM（浏览器对象模型）

  1. JS中推荐使用单引号！

  2. JS中变量可以不声明直接使用，但此变量为全局变量！

  3. JS的变量数据类型只有在程序运行过程中，根据等号右边的值来确定的；

    JS是动态语言，变量的数据类型是可以变化的。

  4. 数字型的最大值：Number.MAX_VALUE

    数字型的最小值：Number.MIN_VALUE

    无穷大：Infinity 无穷小：-Infinity 非数值：NaN (Not a Number)

    isNaN() 方法判断是否是非数字，是数字返回false,否则返回true

    2) instensof 检测前者是否属于后者 【a instensof b 】

  6. 数据类型转换：

    • 转换为字符串：

      toString() alert(num.toString());

      String() 强制转换 alert(String(num));

      加号拼接字符串 alert(num + “我是字符串”);

    • 转换为数字型：

      parseInt(String，[ int radix ]) 函数 parseInt(‘18’);

      String：要转换的值

      [ int radix ]：使用的进制数，默认是”0”十进制

      （若读取的第一个字符是数字，后面其他不是数字的字符都会被去掉；若读取的第一个字符不是数字会出错—>NaN）

      parseFloatt(String) 函数 parseFloatt(‘18.23’);

      （若读取的第一个字符是数字，后面其他不是数字的字符都会被去掉；若读取的第一个字符不是数字会出错—>NaN）

      Number() 强制转换函数 Number(‘12’);

      JS隐式转换（- * /） ‘12’ - 0

      （- 0； * 1； / 1）

    • 转换为布尔型：

      Boolean() 函数 Boolean(‘true’);

      (false —> ‘空’、0、NaN、null、unfefined)

  7. 创建数组：

    var 数组名 = new Array(); 或 var 数组名 = [];

    var 数组名 = new Array(2); // 表示数组长度为2

    var 数组名 = new Array(2，3); // 表示里面数组元素为2，3

    • 检测是否为数组：

    1) instanceof 运算符 —> arr instanceof Array

    2) Array.isArray(参数); —> Array.isArray(arr);

  8. 数组对象：

    • 添加数组元素：

      1) 在数组末尾添加： push();

      arr.push(值,值,…); 返回新数组的长度

      2) 在数组开头添加： unshift();

      arr.unshift(值,值,…); 返回新数组的长度

    • 删除数组元素：

      1) 删除最后一个元素(一次只能删除一个)： pop(); 无参

      arr.pop(); //返回删除的元素

      2) 删除第一个元素(一次只能删除一个)： shift(); 无参

      arr.shift(); //返回删除的元素

    • 数组排序：

      1) reverse(); 颠倒数组元素中的顺序，无参数

      arr.reverse()

      2) sort(); 对数组的元素进行排序，无参数

    • 数组索引方法：

      1) indexOf(); //数组中查找给定元素的第一个索引（从开头开始）

      arr.indexOf( 值 ); //返回第一个满足条件的索引号（数组下标），找不到返回 -1

      2) lastIndexOf(); //在数组中的最后一个索引（从结尾开始）

      arr.lastIndexOf( 值 ); //返回第一个满足条件的索引号（数组下标），找不到返回 -1

      注：索引号（数组下标）不会改变！

    • 数组转换为字符串：

      1) toString();

      arr.toString();

      2) join(‘分隔符’);

      arr.join(); 或 arr.join(‘，’);

    • 其他方法：concat()、slice()、splice() 等

  9. JS中函数的实参和形参个数可以不同，最终取决于形参的个数

    • 若实参个数大于形参—>取到形参个数，多余的实参不参与运算！
    • 若实参个数小于形参—>NaN （形参可以看做是不用声明的变量，若其中一个没有接收值则它为undefined）
    • return 语句之后的代码不会被执行！
    • return 只能返回一个值 (没有return则返回undefined)
      • 若用逗号隔开多个值，以最后一个为准！
      • 若返回多个值，则用数组！（return [a,b,c,d];）

  11. arguments的使用（arguments对象中存储了传递的所有实参，在不确定参数的个数时使用），展示的形式是一种伪数组。

      • 1.它具有数组的 length 属性
      • 2.按照索引的方式进行存储
      • 3.它没有真正数组的一些方法 pop()、push() 等

    • 元素：页面中的所有标签都是元素，DOM中使用element表示

      2) element.getElementsByTagName(‘标签名’) //可以得到这个元素里面的某些标签

      1) document.getElementsByClassName(‘类名’); //根据类名返回元素对象集合

      listener：事件处理函数，事件发生时，会调用该监听函数

      useCapture：可选参数，是一个布尔值，默认是false

  1. 元素偏移量offset系列：

    • offset概述：可以动态的得到钙元素的位置（偏移量）、大小（高度、宽度）等

      注：返回的数值不带单位；定位（position）

    • offset系列属性 作用
      element.offsetParent 返回作为该元素带有定位的父级元素，如果父级都没有定位则返回body
      element.offsetTop 返回元素相对带有定位父元素上方的偏移
      element.offsetLeft 返回元素相对带有定位父元素左边框的偏移
      element.offsetWidth 返回自身包括padding、边框(border)、内容区的宽度，返回数值不带单位
      element.offsetHeight 返回自身包括padding、边框(border)、内容区的高度，返回数值不带单位

    • offset与style区别：

      offset style
      可以得到任意样式表中的样式 只能得到行内样式表中的样式
      获得的数值是没有单位的 style.width获得的是带有单位的字符串
      offsetWidth包含padding+border+width style.width获得不包含padding和border的值
      offsetWidth等属性是只读性，只能获取不能赋值 style.width是可读写属性，可以获取也可以赋值
      想要获取元素大小位置，用offset 想要给元素更改值，用style

  2. 元素可视区client系列：

    • client系列属性 作用

      element.clientLeft 返回元素左边框的大小
      element.clientWidth 返回自身包括padding、内容区的宽度，不含边框，返回数值不带单位
      element.clientHeight 返回自身包括padding、内容区的宽度，不含边框，返回数值不带单位

（五）新增对象Set

数据结构 Set。它类似于数组，但是成员的值都是唯一的，没有重复的值。
Set本身是一个构造函数，用来生成 Set 数据结构。
Set的特点就是该对象里面的成员不会有重复。

1) 基本使用：
let set = new Set(); 得到一个空的Set对象。

2) Set的成员

  • size：属性，获取 set 中成员的个数，相当于数组中的 length
  • add(value)：添加某个值，返回 Set 结构本身。
  • delete(value)：删除某个值，返回一个布尔值，表示删除是否成功。
  • has(value)：返回一个布尔值，表示该值是否为Set的成员。
  • clear()：清除所有成员，没有返回值。

初始化Set的时候，也可以为其传入数组或字符串，得到的Set对象中的成员不会有重复。根据这个特点可以完成数组或字符串去重。
